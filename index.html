Connecting GitHub & Firebase

01

GitHub Repository

Host your code on GitHub to enable auto-deployment and version control.

  1. Create repo "onen-logistics"
  2. Upload this index.html
  3. Enable GitHub Pages (Settings > Pages)
git remote add origin [YOUR_URL]
02

Firebase Cloud

Setup your own database. Go to Firebase Console.

  1. 1. Create Project "ONEN-LOGISTICS"
  2. 2. Add Web App & Copy Config JSON
  3. 3. Enable Firestore (Test Mode)
  4. 4. Enable Auth (Anonymous)

IMPORTANT: Replace the firebaseConfig block in this code with your own.

Security Rules (Production)

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    match /artifacts/onen-studio-v2/public/data/memos/{docId} {
      allow read, write: if true; // Restrict this when you have users
    }
  }
}
